# 2026-06-06 Qwen3.6 2B Q4 Search

## Goal

Check whether a Qwen3.6 2B Q4 GGUF model exists, then download and benchmark it
with the same local-agent procedure if available.

## Sources Checked

- Hugging Face web search for `Qwen3.6 2B GGUF`
- Hugging Face API search through `.venv_hug`
- Unsloth model search results

## Findings

No native Qwen3.6 2B Q4 GGUF repository was found.

The available Unsloth Qwen3.6 GGUF repositories found in this pass are larger
model classes:

- `unsloth/Qwen3.6-27B-GGUF`
- `unsloth/Qwen3.6-27B-MTP-GGUF`
- `unsloth/Qwen3.6-35B-A3B-GGUF`
- `unsloth/Qwen3.6-35B-A3B-MTP-GGUF`

The only 2B-class Qwen3.6-related GGUF candidate found was:

- `khazarai/Qwen3.5-2B-Qwen3.6-plus-Distilled-GGUF`

This is not a native Qwen3.6 2B release. It is tagged as a distilled model with
base model `khazarai/Qwen3.5-2B-Qwen3.6-plus-Distilled`.

Files in that repository:

- `Qwen3.5-2B-Qwen3.6-plus-Distilled-f16.gguf`
- `Qwen3.5-2B-Qwen3.6-plus-Distilled-q8_0.gguf`

No Q4 GGUF file was available there.

## Decision

No download or benchmark was run because the requested target, Qwen3.6 2B Q4
GGUF, was not available.

Keep using `qwen3.5-2b-q4-xl` as the practical Qwen 2B baseline. Recheck Qwen3.6
2B only if Qwen or Unsloth publishes a native 2B/4B-class GGUF or a Q4 quant for
the distilled 2B candidate.
